Connect CRM - Full Stack Management System
Connect CRM is a high-performance MERN Stack application designed for businesses to manage their leads and employee directory efficiently. It features Role-Based Access Control (RBAC), secure authentication, and a clean, responsive UI.

🌐 Live Deployment
Frontend: https://crm-git-main-radheshyamdangis-projects.vercel.app

Backend: https://crmbackend-uv1n.onrender.com


🛠 Tech Stack

Frontend: React.js (Vite), Axios, Lucide React (Icons), React Router 7.

Backend: Node.js, Express.js (v5), MongoDB Atlas.

Security: JWT (JSON Web Tokens), BcryptJS (Password Hashing).

DevOps: Vercel (Frontend), Render (Backend).


🔄 Project Workflow

1. Authentication & Security
Users must register or log in to access the system. The app supports two roles: Admin and Employee.
Encryption: Passwords are never stored in plain text; they are hashed using Bcrypt.
Authorization: Every API request is verified using a JWT token passed in the Authorization header.

2. Lead Management
A central hub to track potential business opportunities.
Create: Add new leads via an interactive modal with fields for Company, Email, and Phone.
Status Tracking: Assign statuses like New, Contacted, or Qualified.
CRUD Operations: Full ability to View, Edit, and Delete leads.

3. Employee Directory
A visual grid of all team members.
Admin Powers: Only Admins can add or delete employees.
Instant Search: Filter the directory in real-time by name or email.
UI Logic: Dynamic avatars generated from the first letter of the employee's name.

⚙️ Installation & Setup
Prerequisites
Node.js (v18+)
MongoDB Atlas Account
